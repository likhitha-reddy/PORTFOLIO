# PORTFOLIO

Personal portfolio website of **G Sai Likhitha Reddy** — Software Developer at Microsoft.

Built as a fast, dependency-free static site with plain **HTML, CSS, and JavaScript**.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Page markup and content |
| `styles.css` | Styling, layout, animations |
| `script.js` | Nav, scroll progress, reveal-on-scroll, typing effect |

## Run locally

Just open `index.html` in a browser, or serve it:

```powershell
python -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (GitHub Pages)

1. Push to the `main` branch.
2. In the repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch → `main` / root**.
3. Your site publishes at `https://likhitha-reddy.github.io/PORTFOLIO/`.
